# HMDA Data (Mortgage rate)

This directory contains intermediate datasets generated from raw inputs for Chapter 2.
All outputs are reproducible using scripts in `code/ch02/01_hmda`.

## Pipeline outputs

### hmda_clean/
  - **Description:** Yearly cleaned HMDA Loan/Application Records (LAR) used as inputs
    for constructing the CBSA-month outstanding mortgage rate panel.
  - **Generated by:** `code/ch02/01_hmda/01_clean_hmda_lar_yearly.R`
  - **Output files (by year, not version controlled):**
    - `hmda_clean_2018.csv`
    - `hmda_clean_2019.csv`
    - `hmda_clean_2020.csv`
    - `hmda_clean_2021.csv`
    - `hmda_clean_2022.csv`
    - `hmda_clean_2023.csv`
    - `hmda_clean_2024.csv`
  - **Time coverage:** 2018–2024

### combined_hmda/
- **Description:** Pooled loan-level HMDA dataset created by combining yearly
  cleaned files and selecting fixed-rate mortgage samples by loan term.
  These datasets serve as direct inputs for constructing monthly outstanding
  mortgage stocks.
- **Generated by:** `code/ch02/01_hmda/02_combined_hmda.R`
- **Output files (not version controlled):**
  - `hmda_30y_clean.csv` — 30-year fixed-rate mortgages (primary analysis sample)
  - `hmda_15y_clean.csv` — 15-year fixed-rate mortgages (kept for reference)
- **Time coverage:** Loans originated from 2018–2024
